# Chapter 2 --- User Experience & Personas

## 2.1 Purpose

This chapter defines the target users, their motivations, expected user
journeys, and the overall user experience principles that guide the
design of OneDate.

The application should feel effortless, personal and emotionally
engaging. Every interaction should reduce planning effort while
increasing excitement for the upcoming date.

------------------------------------------------------------------------

# 2.2 UX Principles

The following principles apply to every screen and interaction.

### Effortless

Users should be able to create an entire date within two minutes.

### Personal

Recommendations must always reflect the user's preferences instead of
generic popularity.

### Emotional

The product should focus on creating anticipation before the date and
preserving memories afterwards.

### Beautiful

Animations should be subtle and premium. The UI should resemble
high-quality consumer apps such as Apple, Airbnb and Spotify.

### Accessible

Navigation, typography, colors and controls must support users of
different abilities.

------------------------------------------------------------------------

# 2.3 User Personas

## Persona A --- The Busy Couple

**Age:** 27--38

Needs: - Doesn't want to spend hours planning. - Wants quality time. -
Appreciates convenience.

Goals: - Plan great dates quickly. - Discover new places. - Save
memories.

Pain Points: - Repeating the same restaurants. - Decision fatigue. -
Lack of inspiration.

------------------------------------------------------------------------

## Persona B --- The New Couple

Needs: - Impress the other person. - Reduce awkward planning. - Receive
conversation ideas.

Goals: - Memorable first dates. - Romantic atmosphere. - Balanced
activities.

Pain Points: - Doesn't know local places. - Fear of choosing poorly.

------------------------------------------------------------------------

## Persona C --- The Explorer

Needs: - Discover hidden gems. - Experience new cultures and cuisines.

Goals: - Unique adventures. - Local experiences. - Authentic
recommendations.

------------------------------------------------------------------------

# 2.4 Core User Journey

## Journey 1 --- Planning a Date

1.  Open OneDate.
2.  Tap **Plan a Date**.
3.  Choose location.
4.  Set budget.
5.  Select date style.
6.  Select cuisine or "Surprise Me".
7.  (Optional) Set duration.
8.  AI generates the complete experience.
9.  Review timeline.
10. Save or regenerate.
11. Start navigation.

------------------------------------------------------------------------

## Journey 2 --- Chemistry Mode

1.  User starts Chemistry Mode.
2.  A shareable invitation link is generated.
3.  Partner answers the same questionnaire independently.
4.  Answers remain hidden until both have finished.
5.  AI compares both profiles.
6.  AI generates:
    -   Compatibility insights
    -   Shared interests
    -   Complementary differences
    -   Personalized date plan

------------------------------------------------------------------------

## Journey 3 --- Memory Book

1.  Date finishes.
2.  User receives a reminder.
3.  Upload photos.
4.  Rate the experience.
5.  Add notes.
6.  Save memories.
7.  AI learns from feedback.

# 2.8 Onboarding Questionnaire

Collect:

-   First name
-   Preferred cuisines
-   Dietary restrictions
-   Favorite activities
-   Indoor / Outdoor preference
-   Alcohol preference
-   Transportation
-   Interests
-   Preferred date styles

Do NOT permanently store the user's budget.

------------------------------------------------------------------------

# 2.9 Emotional Journey

Before the Date: - Excitement - Curiosity - Anticipation

During the Date: - Confidence - Discovery - Fun

After the Date: - Reflection - Gratitude - Memory creation

The interface should reinforce these emotions through copy, imagery and
animations.

------------------------------------------------------------------------

# 2.10 Definition of Success

A successful user experience means:

-   A first-time user can create a complete date without assistance.
-   The AI-generated plan feels personalized.
-   Users want to save their memories after the date.
-   Couples return to plan future experiences instead of uninstalling
    the app.
